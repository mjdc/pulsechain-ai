A Web3 crypto dashboard featuring live simulated market data for Bitcoin and Ethereum. The app integrates Gemini AI to provide real-time market sentiment analysis and technical indicators, combining news signals with price action for concise trading insights.

Live demo: https://pulsechain-ai.vercel.app/

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
